# Thesis AI Website

Marketing and product website for [thesisai.app](https://thesisai.app). Built with Next.js 14, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed to Vercel. Push to `main` to trigger a production deploy.

## Products

- **Thesis AI Platform** — AI-powered investment research (in beta)
- **Long Hold** — iOS app for consistent long-term investing
